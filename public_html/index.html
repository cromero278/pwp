About Me

My name is Cassandra, and I’m a web developer and digital designer based in Albuquerque, New Mexico. I spend my days designing websites and creating various kinds of content for e-commerce brands. I love learning new web technologies and currently enjoy playing around with Javascript and CSS animations. My skills include:

HTML5, CSS, Wordpress, Responsive Web Design, Bootstrap framework, PHP, MySQL, Javascript, Adobe Photoshop & Illustrator

When I’m not making things on the web, I enjoy all things fitness, and spending time outdoors in my lovely home state of New Mexico.
